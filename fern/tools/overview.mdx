Ortavox Agents aren't just for talking; they can *act*. 

By providing custom schemas and descriptions, you can instruct your model to pause the conversation, trigger an external API or database mutation, and then resume the conversation with the newly fetched data.

## Types of Tools
- **Custom Functions:** Provide any valid OpenAPI schema.
- **Transfers:** Handoff an active phone call to a human operator.
- **DTMF:** Allow callers to navigate menus explicitly with keypresses.
