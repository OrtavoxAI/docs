# Web Calls

Ortavox allows you to build seamless, interactive real-time voice experiences directly in the browser using WebRTC technology. 

By leveraging the Ortavox Web SDK, you can quickly integrate an AI Voice Assistant into any web platform without requiring your users to install any plugins or dial a phone number.

## Prerequisites
To get started, ensure you have your API keys ready and have created an Assistant configuration.
