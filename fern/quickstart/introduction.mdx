Ortavox is a lightning block for building real-time voice and communication AI products.

Choose between a comprehensive **Web Calling** interface using WebRTC for browsers or **Telephony (Phone Calling)** for direct inbound and outbound dialing using standard phone networks.

## Getting Started

### 1. Authenticate your API Requests
You'll need an API key to get started. You can generate an API Key using your account dashboard. Ensure all requests to the Ortavox API contain the `Authorization: Bearer <your-api-key>` header.

### 2. Connect Your Communications
Create an inbound phone number, connect to our underlying WebRTC infrastructure, and dispatch the call.

### 3. Build Your Assistant
Equip your AI Agents with robust custom Tools, detailed instructions, and integrate with your existing infrastructure securely.
